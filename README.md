# curium



## Get started


#### Install Starport
[https://github.com/tendermint/starport/blob/develop/docs/1%20Introduction/2%20Install.md](https://github.com/tendermint/starport/blob/develop/docs/1%20Introduction/2%20Install.md)

#### Build and run Curium
```
starport serve
```

#### Run Curium
```
curiumd
```
